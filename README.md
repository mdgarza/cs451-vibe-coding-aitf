# CS451 Vibe Coding – AITF Demo

This project is a simple Node.js and Express application created for **CS451**. It demonstrates a basic server-side rendered web app using **EJS** templates and a small amount of custom CSS styling.

The purpose of this project is to show a working Express server, proper project structure, routing, and GitHub usage — not to build a complex application.

---

## Technologies Used

- Node.js  
- Express.js  
- EJS (Embedded JavaScript Templates)  
- HTML & CSS  
- Git and GitHub

## How to Run the Project

1. Clone the repository:

git clone https://github.com/mdgarza/cs451-vibe-coding-aitf.git

2. Navigate into the project directory:
3. cd cs451-vibe-coding-aitf

4. Install dependencies:

5. npm install: start the server

6. node server.js

7. Open a browser and visit: http://localhost:3000
